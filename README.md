Herramienta de análisis de ciberataques que procesa datos desde CSV, los almacena en SQLite y permite consultas analíticas sobre distribuciones de ataques

- Carga automática de archivos CSV
- Almacenamiento en base de datos SQLite
- Análisis de distribución de tipos de ataque:
  - DDoS (25.08%)
  - Brute Force (25.04%)
  - Port Scan (25.01%)
  - Tráfico Normal (24.88%)

- Python 3.x
- Pandas
- SQLite3
